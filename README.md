# resumo-lab-DIO.tipos_de_servicos_de_nuvem_Azure
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO, sobre Tipos de Serviço de Nuvem na Azure

Conceitos:

#IaaS (Infraestrutura como serviço)#
> + Serviços e armazenamento;
> + FireWalls/ segurança de rede;
> + Planta física, edifício do datacenter (criar uma infraestrutura de TI, pagando conforme uso, alugando servidores, máquinas virtuais, armazenamento, redes e sistemas operacionais de um provedor de nuvem);

#PaaS (Plataforma como serviço)#
> + Além da Infraestrutura citada acima, engloba:
> + Ferramentas p/ desenvolvedores, análise de negócios e gerenciamento de database;
> + Fornece um ambiente p/ a criação, teste e implementação de aplicativos de software, sem focar no gerenciamento de infraestrutura subjacente;

#SaaS (Software como serviço)#
> + Com a Infraestrutura e a Plataforma de serviços, têm-se:
> + Aplicativos e aplicações hospedadas;
> + Os usuários se conectam e usam aplicativos c/ base em nuvem pela internet, Ex: Microsoft Office 365, email e calendário.

*Modelo de Reponsabilidade Compartilhada* 
<br>
#Comparação dos serviços de nuvem#

Iaas
> + flexível; o cliente configura e gerencia o hardware p/ seu aplicativo.

PaaS
> + Desenvolvimento de aplicativos; gerenciamento de plataforma pelo provedor de nuvem;

SaaS
> + Modelo de pagamento conforme uso;
> + Usuários pagam pelo software em assinaturas.
